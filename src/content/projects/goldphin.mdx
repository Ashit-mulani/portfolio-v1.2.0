## <span style={{fontWeight:700, fontSize:"1.65rem", color:"#0369a1"}}>Overview</span>

**GoldPhin** is a <span style={{color:"#ca8a04", fontWeight:600}}>modern crypto wallet dashboard</span> that empowers users to connect a wallet and monitor <span style={{fontWeight:600}}>all statistics</span> in one convenient interface.<br />
No more platform switching — manage your digital assets <span style={{color:"#0369a1", fontWeight:700}}>efficiently</span> from a single, unified dashboard.

---

## <span style={{fontWeight:700, fontSize:"1.3rem", color:"#be123c"}}>Problem & Pain Points</span>

Managing crypto across many wallets and chains is <span style={{fontWeight:500, color:"#be123c"}}>fragmented and a chore</span>.

Users must:

- <span style={{fontWeight:500}}>Check balances</span> on one app
- <span style={{fontWeight:500}}>Track tokens</span> elsewhere
- <span style={{fontWeight:500}}>Swap assets</span> on a DEX

<span style={{fontStyle:"italic", color:"#64748b"}}>The result: inefficiency and poor UX.</span>

---

## <span style={{fontWeight:700, fontSize:"1.25rem", color:"#15803d"}}>Solution</span>

GoldPhin gives you:

- <span style={{fontWeight:500}}>Seamless wallet connection</span>
- <span style={{fontWeight:500}}>Real-time portfolio snapshots</span>
- <span style={{fontWeight:500}}>One-click token swaps</span> <span style={{fontSize:"0.85em", background:"#fef9c3", color:"#a16207", padding:"2px 6px", borderRadius:"6px"}}>1inch integration</span>
- <span style={{fontWeight:500}}>Multi-chain asset support</span>
- <span style={{fontWeight:500}}>Speedy live data fetching</span>

<p style={{marginTop:"0.5em"}}><span style={{fontStyle:"italic", color:"#64748b"}}>All elegantly presented in a single dashboard UI.</span></p>

---

## <span style={{fontWeight:700, fontSize:"1.15rem", color:"#3730a3"}}>Core Features</span>

- **Multi-chain token overview** — Instantly check balances across all supported blockchains
- **In-dashboard token swap** — Fast, secure trades powered by 1inch
- **Market insights** — Real-time price and performance data
- **Portfolio tracking** — Visualize gains, losses and allocation history
- **Secure connections** — Effortless, private wallet authentication
